00222 /*
00223  These arguments get replaced with the Local_Notification_sprintf
00224  and other such functions found in notifications.qc to supply data
00225  from networked notifications to their usage in sprintf... It
00226  allows for more dynamic data to be inferred by the local
00227  notification parser, so that the server does not have to network
00228  anything too crazy on a per-client/per-situation basis.
00229 
00230  Pay attention to the CSQC/SVQC relations, some of these are redefined
00231  in slightly different ways for different programs, this is because the
00232  server does a more conservative approach to the notifs than the client.
00233 
00234  All arguments are swapped into strings, so be sure that your
00235  sprintf usage matches with proper %s placement.
00236 
00237  Argument descriptions:
00238     s1-s4: string arguments to be literally swapped into sprintf
00239     s2loc: s2 string of locations of deaths or other events
00240     s3loc: s3 string of locations of deaths or other events
00241     f1-f4: float arguments expanded into strings to be swapped into sprintf
00242     f1p2dec: f1 float to string with 2 decimal places
00243     f2p2dec: f2 float to string with 2 decimal places
00244     f2primsec: f2 float primary or secondary selection for weapons
00245     f3primsec: f3 float primary or secondary selection for weapons
00246     f1secs: count_seconds of f1
00247     f1points: point or points depending on f1
00248     f1ord: count_ordinal of f1
00249     f1time: process_time of f1
00250     f1race_time: mmssss of f1
00251     f2race_time: mmssss of f2
00252     race_col: color of race time/position (i.e. good or bad)
00253     race_diff: show time difference between f2 and f3
00254     missing_teams: show which teams still need players
00255     pass_key: find the keybind for "passing" or "dropping" in CTF game mode
00256     frag_ping: show the ping of a player
00257     frag_stats: show health/armor/ping of a player
00258     frag_pos: show score status and position in the match of a player
00259     spree_cen: centerprint notif for kill spree/how many kills they have
00260     spree_inf: info notif for kill spree/how many kills they have
00261     spree_end: placed at the end of murder messages to show ending of sprees
00262     spree_lost: placed at the end of suicide messages to show losing of sprees
00263     item_wepname: return full name of a weapon from weaponid
00264     item_wepammo: ammo display for weapon from string
00265     item_centime: amount of time to display weapon message in centerprint
00266     item_buffname: return full name of a buff from buffid
00267     death_team: show the full name of the team a player is switching from
00268     minigame1_name: return human readable name of a minigame from its id(s1)
00269     minigame1_d: return descriptor name of a minigame from its id(s1)
00270 */
