00033 // modal dialog controller
00034 // handles a stack of dialog elements
00035 // each element can have one of the following states:
00036 //   0: hidden (fading out)
00037 //   1: visible (zooming in)
00038 //   2: greyed out (inactive)
00039 // While an animation is running, no item has focus. When an animation is done,
00040 // the topmost item gets focus.
00041 // The items are assumed to be added in overlapping order, that is, the lowest
00042 // window must get added first.
00043 //
00044 // Possible uses:
00045 // - to control a modal dialog:
00046 //   - show modal dialog: me.showChild(me, childItem, buttonAbsOrigin, buttonAbsSize, 0) // childItem also gets focus
00047 //   - dismiss modal dialog: me.hideChild(me, childItem, 0) // childItem fades out and relinquishes focus
00048 //   - show first screen in m_show: me.hideAll(me, 1); me.showChild(me, me.firstChild, '0 0 0', '0 0 0', 1);
00049 // - to show a temporary dialog instead of the menu (teamselect): me.hideAll(me, 1); me.showChild(me, teamSelectDialog, '0 0 0', '0 0 0', 1);
00050 // - as a tabbed dialog control:
00051 //   - to initialize: me.hideAll(me, 1); me.showChild(me, me.firstChild, '0 0 0', '0 0 0', 1);
00052 //   - to show a tab: me.hideChild(me, currentTab, 0); me.showChild(me, newTab, buttonAbsOrigin, buttonAbsSize, 0);
